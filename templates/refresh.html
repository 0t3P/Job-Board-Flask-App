Refresh Jobs Data

How to Update Jobs

The job listings are scraped from various websites. To refresh the data, run the scraper script.

cd C:\Python\desktop_python_files\other_script\mobile
python run_scrapers.py

python run_scrapers.py --spiders onlinejobs remoteok freelancer

python run_scrapers.py --all

python run_scrapers.py --pages 5
Available Scrapers
onlinejobs - OnlineJobs.ph
remoteok - RemoteOK
weworkremotely - We Work Remotely
freelancer - Freelancer.com
guru - Guru.com
peopleperhour - PeoplePerHour
indeed - Indeed
hubstaff - Hubstaff Talent
After Running the Scraper
1 Jobs will be saved to scraped_jobs.json in the parent directory
2 Run update_jobs.bat to copy data into the jobs_flask folder
3 Refresh this website to see the new jobs
4 No need to restart the Flask server
Quick Update: Double-click update_jobs.bat or manually copy scraped_jobs.json from the parent directory into the jobs_flask folder.
